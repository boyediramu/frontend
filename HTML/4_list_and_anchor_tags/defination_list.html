INTRODUCTION TO HTML

HTML:

HTML (HyperText Markup Language) is the standard, foundational markup language used to create and structure web pages and applications. It uses a system of tags and elements to define components like headings, paragraphs, links, and images, which web browsers interpret to render content. It is not a programming language but a structural one.

History of HTML:

Html is invented by Tim Berners Lee in 1991 He designed it as a way to share and link documents over then internet

Tags:

Tags are predefined words which helps to build the static structure of a web page. Tags are written by using angle brackets open and close

Types of Tags:

  1. Single Tag: It is a self-closing tag

    • hr
    • br
    • img
    • input

  2. Paired Tag: Both opening and closing tags

    • P
    • pre
    • heading tags

Elements:

Combination of opening tag, closing tag and content

Text formatting Tags:

Formatting tags are used to change the appearance of text like making it bold, italic, underlined, small, big, marked, etc.

Examples:

  • (b) Bold text
  • (strong) strong text
  • (i) Italic text
  • (em) emphasized text
  • (u) underline text
  • (mark) marked text
  • (small) small text
  • (big) big text
  • (del) deleted text
  • (ins) inserted text
  • (sup) superscript
  • (sub) subscript

Attributes:

Attributes are one of the core component in html It will provide the additional information about an Html tag or element. Attributes are always written in open tag

Syntax: Attribute name=”value”

Types of Attributes:

  1. Global attributes

    • class
    • id
    • style
    • title

  2. Element specific attributes

    • href
    • src
    • width
    • height

Marquee Tag:

The marquee tag is used to create scrolling (moving) text or images inside a webpage.
